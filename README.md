# 摸鱼日报 🐟

> 工作再累，一定不要忘记摸鱼哦！

一个有趣的摸鱼工具，以复古报纸风格展示距离下一个假期还有多久，提醒打工人适时休息。

## ✨ 功能特性

### 📰 摸鱼日报主页
- **复古报纸风格设计**：采用传统报纸排版，增强视觉吸引力
- **假期倒计时**：自动计算距离下一个法定节假日和周末的天数
- **疫情记录**：记录自新冠疫情爆发以来的天数
- **双栏布局**：优雅的双栏内容展示，提升阅读体验
- **期号自动计算**：从 2024-01-01 开始自动计算每日期号
- **响应式设计**：完美适配桌面端和移动端

### 💝 表白页面
- 浪漫的心形跳动动画
- 温馨的表白文案展示
- 适合向心仪的 Ta 表达心意

### 🎯 核心功能
- 支持 2022-2025 年中国法定节假日数据
- 自动识别调休和补班日期
- 实时更新当前日期和时间
- 优雅的加载动画

## 🛠️ 技术栈

### 后端
- **Spring Boot** 2.7.1 - Java 应用框架

## 🚀 快速开始

### 环境要求
- JDK 11 或更高版本
- Maven 3.x

### 安装步骤

1. **克隆项目**
```bash
git clone <repository-url>
cd myb
```

2. **编译项目**
```bash
mvn clean package
```

3. **运行项目**
```bash
mvn spring-boot:run
```

或直接运行打包后的 JAR 文件：
```bash
java -jar target/myb-0.0.1-SNAPSHOT.jar
```

4. **访问应用**
- 摸鱼日报主页：http://localhost:8090/
- 表白页面：http://localhost:8090/love.html

## 🎨 设计理念

### 报纸风格
采用复古报纸设计，包含：
- 经典的报头设计（深色背景 + 金色点缀）
- 装饰性边框和角落元素
- 传统的双栏排版
- 优雅的衬线字体

### 用户体验
- 简洁直观的信息展示
- 流畅的加载动画
- 清晰的视觉层次
- 舒适的阅读体验

## 📝 开发说明

### 假期数据管理
假期数据以 JSON 格式存储在 `MybController.java` 中：
- `holiday2022`：2022 年假期数据
- `holiday2024`：2024 年假期数据
- `holiday2025`：2025 年假期数据

如需添加新年份的假期数据，请按相同格式添加到 `holidayMap` 中。

### 代码规范
- 使用 4 空格缩进
- 遵循 Java 代码规范
- 前端代码保持简洁易读
